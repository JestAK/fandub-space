# About
Web-app for fandub community to share their work and interact with each other.

# How to run

1. Clone the repository
2. Go to the frontend directory `cd frontend`
3. Install dependencies `npm install`
4. Build the app `npm run build`
5. Run the app `npm run start`
6. Go to the backend directory `cd ../backend`
7. Install dependencies `npm install`
8. Run the app `npm run start`
